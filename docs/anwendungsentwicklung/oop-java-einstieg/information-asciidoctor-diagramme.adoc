== AsciiDoc

https://de.wikipedia.org/wiki/AsciiDoc[AsciiDoc] ist eine
https://de.wikipedia.org/wiki/Kategorie:Vereinfachte_Auszeichnungssprache[vereinfachte Auszeichnungssprache], wie z.{nbsp}B.
auch https://de.wikipedia.org/wiki/Markdown[Markdown].

Ein AsciiDoc-Dokument wird mit einem Texteditor wie TextMate, GEdit,
Notepad++, Vim, Emacs oder Sublime Text verfasst --
Textprozessoren wie Word oder Writer sind nicht geeignet.
Nach dem Speichern (mit der Dateiendung `adoc`)
wird ein AsciiDoc-Dokument mit einer Software
in ein Ausgabeformat (PDF, HTML5, Docbook usw.) übersetzt.
