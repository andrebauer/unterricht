== Lichtsteuerung für das SmartHome

Die HomeSoft AG produziert Produkte im Bereich "SmartHome" und hat Sie
beauftragt, eine Steuerung für Schalter und Lampen zu entwickeln.

Schalter und Lampen sollen als objektorientierte
Software-Module entwickelt werden, so dass nach Bedarf Schalter und
Lampen miteinander verbunden werden können.

=== Aufgabe 1

Entwerfen Sie jeweils ein UML-Objekt-Diagramm für einen Schalter und eine Lampe

[plantuml, diagram-objects, svg]
....
object Schalter
object Lampe 
....

Erstellen Sie ein UML-Klassen-Diagramm für einen Dimmer mit den Stufen 0--3.
